## 📚 실습 내용

### 1. 명령 창 사용하기

- `pi` 상수 출력 및 포맷 변경 (`format long`, `format short`)
- 기본 연산 (`*`, `^`)
- 변수 확인 (`who`, `whos`)

### 2. 스칼라, 벡터, 행렬 생성하기

**스칼라**
- 단일 숫자 값 할당

**벡터**
- 행 벡터: `[1 2 3 4 5]`
- 열 벡터: `[1; 2; 3; 4; 5]` 또는 `[1 2 3 4 5]'`
- 등간격 벡터: `1:5`, `1:0.5:3`
- `linspace`, `logspace` 함수 활용

**행렬**
- 일반 행렬: `[1 2 3; 4 5 6; 7 8 9]`
- 특수 행렬: `zeros`, `ones`, `eye`, `diag`

### 3. 행렬 계산하기

**벡터의 곱**
- 내적 (Inner Product): 행벡터 × 열벡터
- 크로스곱 (Cross Product): `cross(a, b)` (3차원 벡터)
- 외적 (Outer Product): 열벡터 × 행벡터

**행렬의 곱**
- 행렬 곱: `A*B`
- 행렬 제곱: `A^2` (정방 행렬)
- 원소별 곱: `A.*B`
- 원소별 제곱: `A.^2`

### 4. 행렬 인덱싱과 편집

**인덱싱 방법**
- 위치 기준 인덱싱: `A(3,2)`, `A(1:3, 2:4)`
- 선형 인덱싱: `A(6)`
- 논리형 인덱싱

**행렬 편집**
- 요소 수정: `A(2,3) = 100`
- 행/열 수정: `A(1,:) = [10 20 30]`
- 행렬 결합: `[A B]`, `[A; B]`

### 5. 함수

**내장 함수**
- 수학 함수: `log`, `log10`, `sqrt`, `round`, `ceil`, `floor`
- 통계 함수: `max`, `min`, `mean`, `sum`, `prod`

**입출력 함수**
- `disp`: 간단한 출력
- `fprintf`: 형식화된 출력
- `input`: 사용자 입력

### 6. 구조 프로그래밍

**스크립트**
- `.m` 파일로 여러 명령을 순차 실행

**조건문**
- `if-elseif-else-end` 구조
- 학점 계산 예제
- 에러 처리: `error()` vs `disp()`

**반복문**
- `for` 문: 정해진 횟수 반복
- `while` 문: 조건 기반 반복
- 제어 키워드: `break`, `continue`

**사용자 정의 함수**
- 기본 함수 작성
- 부함수 (Subfunction)
- 무명 함수 (Anonymous Function): `@(x) x^2`
- 함수 핸들 전달

## 🤔 자주 묻는 질문

**Q: MATLAB의 인덱스는 0부터 시작하나요?**  
A: 아니요, MATLAB은 1부터 시작합니다. (C/Python과 다름)

**Q: `A*B`와 `A.*B`의 차이는 무엇인가요?**  
A: `A*B`는 선형대수 행렬 곱셈, `A.*B`는 같은 위치의 원소끼리 곱하는 연산입니다.

**Q: 세미콜론(`;`)은 언제 사용하나요?**  
A: 1) 명령 끝에 붙여서 출력 억제, 2) 행렬에서 행 구분, 3) 벡터에서 열 구분
